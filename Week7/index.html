Week 07

Notes:

  • The length property returns the number of parameters that a function has.
  • An IIFE function is an anonymous function that is called as soon as it is defined. To create an IIFE function we must put parentheses at the end of the function definition.
  • With the temp keyword, we can create temporary variables that are used just one time and then disappear.
  • A recursive function is a function that calls itself when a condition is true.
  • We can use promises to simplify the process of using multiple callbacks.
  • There are functions that allow us to return other functions.
  • The generators are special functions used to produce iterators that maintain the state of a value.
  • Using ajax we can asynchronously connect to a server. Refresh pages without the need to reload them
  • With the Fetch API we are able to fetch the URL we are looking for and handle the responses we wish to give.
  • With .catch we provide a response if an error occurs with the server
  • Response has more properties, some are headers, URL, redirected, and type.
