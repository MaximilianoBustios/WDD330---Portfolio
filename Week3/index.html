Week 03

Notes:

  • With "this" the methods can refer to the object
  • "this" can be used in different contexts so we have to know how it is being used.
  • You can replace "this" with other statements but it is important to use it and know how it works.
  • If an object has functions inside, they are called "methods".
  • Objects have getters and setters, these are used to get and change the values ​​of their data.
  • Thanks to the DOM we can have elements that directly access HTML elements that contain information.
  • With the innerHTML property, we can use HTML in the DOM.
  • We call events all the actions that we have to carry out when the user interacts with the elements of the website.
  • There are two ways to propagate an event, "capturing" and "bubbling". One acts on the element it was acted on and the other acts from the root element. Both can be used.
  • The parameter of our functions should not always be called "event", it can be different.
