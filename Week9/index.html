Week 09

Notes:

  • BOM helps us to access multiple features of our browser
  • Thanks to this we can know which browser is used to view a page
  • We can also know the dimensions of the page and the pages that were visited before and the current page.
  • Through the window object, we can access its methods and properties.
  • Global variables created within the program belong to the window object.
  • parseInt() and isNaN() are global methods, so they are methods of the window object.
  • The Navigator object contains information about the browser that is being used.
  • The userAgent property displays information about the operating system the user is using. Although this property is still supported by all browsers, it has been deprecated.
  • The window.location property is an object that contains the URL information of the current page.
  • window.location.href returns the full url as a string.
  • The hostname property returns a string that describes the domain of the current URL.
  • window.location has the following methods:
    1. The reload() method can be used to force a reload of the current page. If given a parameter of true, it will force the browser to reload the page from the server, instead of using a cached page.
    2. The assign() method can be used to load another resource from a URL provided as a parameter, for example:
  • With window.history we can access information about any previously visited page from the current page.
  • With the window.open() method you can open a new window.
  • The close() method can be used to close a window.
  • With window.screen we can know the dimensions in pixels of the screen. With the height and width properties, we can know the height and width.
  • When we use document.write() this replaces the entire HTML document with the string we input into the method.
  • We can also create and reassign cookies using document.cookie
  • The data- attribute is a way to embed data in a web page using custom attributes that are ignored by the browser. They are also private to a page, so they are not intended to be used by an external service; their sole purpose is to be used by a JavaScript program.
  • With the dataset property we can access the data attributes.
  • APIs give us access to hardware, such as cameras, batteries, geolocation, and the graphics card.
  • localStorage and sessionStorage are properties of the window object.
