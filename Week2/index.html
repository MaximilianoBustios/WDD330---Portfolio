Notes:

  • Javascript can convert a string to a number when it is being used in some mathematical operation. This is called type coercion.
  • Undefined variables are those that have not been assigned a value.
  • Null is a value ns "no value"
  • Undefined and Null are similar but they don't mean the same
  • When we work with Arrays we must always be aware that the references we use point to an existing value. This is to avoid storing values that are no longer needed.
  • The sets are a structure that can only store unique values, that is, not repeated.
  • Callback: a function that is passed as an argument to another.
